# es6-media-list-app

Simple media list application written in ES6. Used technologies: BabelJS, Karma, Jasmine.

## Run application

### Install dependencies

```
npm i
```

### Build application
```
npm run build
```

### Run 
```
http-server -c-1
```

## Tests

### Run tests
```
npm run tests
```

### Checkout test coverage
```
cd coverage/phantomjs
http-server -c-1
